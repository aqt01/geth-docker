# geth-rinkeby-docker

> Run a [geth](https://github.com/ethereum/go-ethereum) node that syncs up with [rinkeby](https://www.rinkeby.io/) testnet in a [Docker](https://www.docker.com/) container.

# Usage

```bash
docker-compose up
```

## Ports

RPC URL

```
http://{docker_ip}:8545/
```

WebSocket URL

```
http://{docker_ip}:8546/
```
